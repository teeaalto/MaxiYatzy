This is a game of MaxiYatzy made by Tuomas Aalto.

At the moment, the game is only runnable in command prompt. To run the
game, compile the source code with e.g. the Scala Build Tool 
https://www.scala-sbt.org/

The specific rules of the game are explained within the game. Note that
there are plenty of MaxiYatzy rule variants out there, so be sure to check
what are the rules used in this game.
